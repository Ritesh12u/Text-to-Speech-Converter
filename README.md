# Text to Speech Converter

A modern web application that converts written text into natural, human-like speech using the Web Speech API. Built with HTML, CSS, and JavaScript, this app runs entirely in the browser, requiring no backend or external APIs.

## Features

- Real-time conversion of text to speech.
- Multiple voice options with language and accent selection.
- Adjustable speech rate and pitch for personalized audio output.
- Playback controls: Speak, Pause, Resume, Stop.
- Responsive, elegant UI with glassmorphism design.
- Works on all modern browsers supporting speech synthesis.

## How It Works

The application uses the browser’s built-in text-to-speech capabilities accessed via JavaScript’s SpeechSynthesis API. Users type or paste text, select voice and audio settings, and listen to the spoken output instantly. Speech playback can be controlled with pause, resume, and stop functions.

## Usage

1. Clone or download the repo.
2. Open `index.html` in a modern web browser.
3. Enter your text in the textarea.
4. Choose your preferred voice from the dropdown.
5. Adjust rate and pitch sliders as desired.
6. Click the "Speak" button to hear the text spoken aloud.

## Use Cases

- Accessibility aid for visually impaired users.
- Language learning and pronunciation practice.
- Quick narration for content creators.
- Multitasking while listening to textual content.

## Technologies Used

- HTML5 for structure.
- CSS3 with glassmorphism styling for a polished UI.
- JavaScript leveraging the Web Speech API for speech synthesis.
